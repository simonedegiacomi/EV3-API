Detailed Description

Use these constants to configure the desired regulation mode for the specified motor(s): none, speed regulation, multi-motor synchronization. For the EV3 only the speed regulation constant has any effect. If you set power instead of speed then no regulation is in effect (equivalent to using OUT_REGMODE_IDLE with the NXT.
